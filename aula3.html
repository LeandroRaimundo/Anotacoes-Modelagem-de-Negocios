STAKEHOLDER E HISTÓRIA DE USUÁRIOS

STAKEHOLDERS

  1. Stakeholder é uma pessoa ou organização que tem influência direta ou indireta nos requisitos de um sistema.
  2. Basicamente é quem organiza os projetos de uma empresa.
  3. Stakeholders são provavelmente a maior fonte de requisitos.
- Precisamos saber os requisitos de uma sistema da empresa.

- Diferentes stakeholders podem ser responsáveis por cada setor de uma empresa.

Stakeholders Internos

  • Employees
  • Manager
  • Owners

Stakeholders Externos

  • Suppliers
  • Society
  • Government
  • Creditors
  • Shareholders
  • Customers

Objetivos da Engenharia de Requisitos

  • Conhecer os requisitos relevantes e estabelecer um consenso entre os stakeholders a respeito dos requisitos.
  • Compreender e documentar as expectativas e necessidades dos stakeholders, especificar e gerenciar os requisitos para minimizar os riscos.

    • - Basicamente entender o requisito do cliente e fazer ele ter um objetivo.

Personas

  • Personas são personagens que representam seu cliente.
  • Criar um personagem com as características relacionadas ao perfil do usuário.
  • Um produto pode ter mais de uma persona.
  • Ajuda na identificação dos papéis.
    • - Ex: RH, Gerente, Funcionário.

Histórias de usuário são informais

  • NÃO substituem a documentação do projeto, nem os requisitos funcionais
  • User stories servem para representar necessidades reais e práticas do usuário.
  • User stories podem sofrer alterações ao longo do projeto.
  • São “narrações” de uma funcionalidade pelo ponto de vista do usuário.

Devem representar valor de negócio

  • Deve saber o motivo das user stories existirem.
  • Validar se a funcionalidade é realmente necessária.
  • Deixar claro o valor do negócio em cada use story.

Identifique os épicos

  • Épico: são histórias de usuários que demandam muitas outras ações
    • - Ex: "Como usuário gostaria de pagar minha compra para que possa receber meus produtos."

  • Dividir os Épicos, é uma excelente maneira de encontrar boas novas histórias.
